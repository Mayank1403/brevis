*Requirements*
```
Python version-3.6.7
Flask version - 0.12
npm   version - 6.14.5
```

*Dependencies Installation*
```
cd frontend;
```
```
npm install
```

*Starting the Application*

To start the server type the following  commands :
```
cd backend;
```
```
python3 prog.py
```
Open a new terminal
To start the client type the follwoing commands :
```
cd frontend;
```

```
npm start
```
